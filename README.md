# Crime-Data-OLAP-Analysis
A Data Warehousing and Data Mining (DWDM) practical project implementing OLAP operations using Python and Pandas. Includes data cleaning, handling missing values, and multidimensional data analysis on a crime dataset using Jupyter Notebook.
DWDM OLAP Practical Project

This project demonstrates OLAP operations (Slice, Dice, Roll-up, Drill-down, Pivot) using Python and Pandas on a crime dataset.

It includes:
- Data cleaning
- Handling missing values
- Multidimensional data analysis
- Jupyter Notebook implementation
